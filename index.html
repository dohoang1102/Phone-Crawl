Phone-Crawl by MaoDaAmargura

Rogue-Like Dungeon Crawl game for the iPhone.

Search changing dungeons, best daunting foes, and collect powerful weapons to earn your place amongst the great heros of Tin'Foyel!

Install

I don't know of a way to run binaries on the iPhone, so you'll have to open this in XCode and run it in the Simulator.
